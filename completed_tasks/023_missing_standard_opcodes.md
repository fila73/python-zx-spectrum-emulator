# Task: Implement Missing Standard Opcodes

1. Implement `0x07` (RLCA): Rotate Accumulator Left.
2. Implement `0x0F` (RRCA): Rotate Accumulator Right.
3. Implement `0x17` (RLA): Rotate Accumulator Left through Carry.
4. Implement `0x1F` (RRA): Rotate Accumulator Right through Carry.
5. Implement `0xF1` (POP AF): Pop AF from stack.
6. Verify implementation with unit tests.
