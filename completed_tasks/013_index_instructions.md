# Task: Index Instructions (Prefix DD/FD)

1. Implement IX/IY Register Access and Management.
2. Implement Indexed Addressing (`LD r, (IX+d)`, `LD (IX+d), n` etc.).
3. Implement Indexed Arithmetic (`ADD IX, ss`, `INC IX` etc.).
4. Implement DD/FD prefixed bit instructions (DDCB/FDCB).
5. Verify with tests.
